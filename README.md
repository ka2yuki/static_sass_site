# What this
Boilerplate for Scss

# How to use
## Ruby
```sh
# This COMMAND is installed to Path at '/Users/[yourName]/.rbenv/shims/' if when be used 'rbenv'.
gem install sass
# sass
sass sass/main.scss:css/main.css --style compressed 
# scss
sass sass/main.scss:css/main.css --style compressed 
```
or 
## Node.js
```sh
npm run sass
```
