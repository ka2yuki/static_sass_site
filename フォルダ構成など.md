
# Scssのコンパイル手順
Node.jsを使用します。
###### ご自身のPCに nodejs が必要です。

1. インストールし、
2. コマンド走らせます。

```sh
npm i # install
npm run css-build # 初回時.
npm run css-watch # それ以降の変更検知。
```
# フォルダ説明
- components: 管理しやすい様にパーツに分けています。
- funcs: 奇数か偶数チェックのシンプルなメソッドのみです。
